Elastic 영상 모음

Elastic 한국 지사 및 커뮤니티에서 제작된 Elastic 활용 영상을 모아놓은 페이지 입니다. 모든 영상에 대한 권리는 작성자와 Elastic 사에 있으며 본 영상을 사내 교육, 학교 수업, 개인 기록물 등에 활용하고자 하시는 분들은 출처를 밝히고 사용 여부를 알려 주시기 바랍니다.

데이터 수집 및 시각화 데모

(2019) 서울시 지하철 대시보드 만들기

서울시 공공데이터를 이용하여 서울시 지하철 승하차 인원 대시보드를 만드는 방법을 설명합니다.

(2021) 처음부터 시작하는 Elastic

Elastic 사와 협력을 맺은 대학교 학생들의 학습을 위해 처음 제작이 된 영상으로 Elastic 을 시작하는 다른 분들에게도 도움이 되도고자 커뮤니티에도 공개되었습니다. GCP 클라우드에서 리눅스 서버 생성 부터 Elasticsearch 클러스터 구성, 데이터 수집, Kibana 시각화 까지의 전반적인 내용이 총 29개 영상으로 구성되어 있으며, 각 영상의 길이는 10~20분, 전체 분량은 약 7시간 15분 입니다. 영상에서 설명하는 Elastic Stack 버전은 7.11 입니다.

과정 소개 및 시스템 준비

GCP 에서 인스턴스를 생성하는 방법과 기본적인 linux 사용법에 대해 설명합니다.

Stack 설치 및 환경 설정
