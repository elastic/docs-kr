Elasticsearch 는 각 노드들 별로 실행될 설정들을 적용함으로써 <DocLink id="esG7-02-03-03-node-config" text="노드들의 역할을 나누"/>거나 클러스터의 속성을 결정하게 됩니다. Elasticsearch 의 실행 환경을 설정하는 방법은 크게 2가지가 있습니다. 홈 디렉토리의 config 경로 아래 있는 파일들을 변경하거나 시작 명령으로 설정하는 방법입니다. config 경로 아래의 파일들에서는 다음과 같은 설정들이 가능합니다.

* <DocLink id="esG7-02-03-01-jvm" text="jvm.options"/> - Java 힙메모리 및 환경변수
* <DocLink id="esG7-02-03-02-es-yml" text="elasticsearch.yml"/> - Elasticsearch 옵션
* log4j2.properties - 로그 관련 옵션

그 외에도 Elasticsearch 를 처음 실행할 때 <DocLink id="esG7-02-03-04-cmd" text="-E 커맨드"/> 라인 명령을 통해서도 가능합니다.
