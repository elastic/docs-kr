Gobld

What is Gobld

Gobld (pronounced: ɡɒb(ə)l di) is a dynamic agent launcher for Buildkite.
It listens for Webhook events from Buildkite for scheduled jobs, and launches an agent based on the provider default configuration and any supplied dynamic configuration tags.
Once the job has finished the launched agent will be destroyed.

Gobld can launch agents in any of the supported providers and has a plugin architecture to easily add new providers.

You can see a pretty dashboard of what this looks like in ci-stats.elastic.co.

How can I connect to an immutable agent via ssh?

Our docs about connecting to Linux instances include details about how to do this.

What metrics are available?

Gobld exposes a number of useful metrics about Jenkins and cloud providers using a Prometheus format end-point on a configurable port, with the default being 8080.

More details on the metrics, what they record, etc can be found here: metrics.md
